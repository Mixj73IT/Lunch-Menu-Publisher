# Core UX Rules (Locked)

## Layout
- Center: month calendar (always visible)
- Left panel: Entrée tile library
- Right panel: Side tile library
- Panels are full‑height and collapsible
- Calendar is the primary focus at all times

## Panels
- Default: panels open
- Collapsed state does NOT persist between sessions
- Panels collapse individually

## Tile Libraries
- Tiles shown in a grid (default 2 columns)
- Tiles are draggable
- Tile order is user‑customizable via drag
- Tile order persists across sessions
- Tiles are NOT edited in the library

## Conditional Density
- Panels may switch to a 3‑column grid
- Only if the opposite panel is collapsed
- Only if enabled in Settings
- Never automatic
- Never both sides simultaneously

## Editing
- Drag is primary interaction
- Clicking a day enables inline editing
- Typing is allowed as a fallback
- No modal dialogs for editing days
