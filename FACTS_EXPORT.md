# FACTS Export

## Purpose
Provide a plain‑text copy/paste format.

## Rules
- Plain text only
- One line per weekday
- No weekends
- No formatting
- No dependency on PDF styling
- FACTS export lines must include an explicit day‑of‑week and numeric date (e.g., “Mon 9/15”) and include only weekdays with active lunch service.


## FACTS is secondary.
PDF quality takes precedence.
