# PDF Design Intent

## Priority
The PDF is the primary product.

Parents and students will read it.
It represents the school.

## Tone
- Christian
- Calm
- Reverent
- Intentional
- Not flashy
- Not playful

## Typography
- Month title: serif
- Verse text: script
- Verse reference: small caps serif
- Calendar content: clean, legible font

## Rules
- Script font ONLY for verse text
- No background images
- No freeform layout
- Weekends narrower than weekdays
- Grid must print cleanly on office printers

## Preview Mode
- Calendar only
- No side panels
- No editing controls
- Matches PDF exactly
