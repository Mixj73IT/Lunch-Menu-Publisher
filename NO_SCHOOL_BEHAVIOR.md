# NO SCHOOL Behavior Specification

## Definition
“NO SCHOOL” is a visual state, not text entry.

## Weekends
- Always NO SCHOOL
- Narrower column width
- Darker neutral shaded background
- Day number visible in white
- No text of any kind inside the cell

## Weekday NO SCHOOL
- Normal column width
- Same darker neutral shaded background
- Day number visible in white
- Small‑caps “NO SCHOOL” text centered
- No menu content displayed

## Visual Rule
If a cell is shaded, the day number MUST invert to white.

## Color & Accessibility
- Meaning must rely on luminance (light/dark), not color
- Must survive black‑and‑white printing
- Must be clear to color‑blind users
